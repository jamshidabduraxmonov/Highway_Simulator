## Overview

This project explores how navigation systems and traffic simulators work under the hood.

The goal is to create a vehicle capable of navigating through a real road network while obeying traffic rules and responding to environmental conditions.

---

## Features

- Interactive map powered by Leaflet
- Real road coordinate visualization
- Vehicle movement simulation
- Smooth coordinate interpolation
- Destination selection by clicking on the map
- Route visualization
- Scalable architecture for future traffic systems

---

## Installation

```bash
npm install
npm run dev
```

---

## Long-Term Goals

The vehicle should eventually be able to:

1. Select any destination on the map
2. Calculate a valid route
3. Follow road geometry
4. Obey traffic lights
5. React to congestion
6. Reach the destination realistically
